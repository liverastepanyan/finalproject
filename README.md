BookSwap — цифровая платформа для обмена, продажи и безвозмездной передачи книг среди студентов. Проект создан в рамках курса «Python. Анализ данных в маркетинге».

Аналитическая задача проекта: • Выявить факторы, влияющие на успешность сделки на платформе • Построить классификационную модель для предсказания исхода сделки • Провести лингвистический анализ отзывов пользователей • Оценить качество сервиса через данные и сформулировать маркетинговые выводы
